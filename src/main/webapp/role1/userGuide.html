Daycare Attendee's

Contact Information User's Manual

PURPOSE

Here at Kiddie Kollege Daycare we access our attendee's information through a St. Paul's Lutheran School web service. This allows our children's contact information to be more accessible as needed. Information available to us includes: Parents, Emergency Contacts, and Physicians. You'll have access to the contacts First Name, Last Name, Address1, Address2, City, State, Zip, Email, Cell Phone, Work Phone, and their Relationship to the child.

Interface

Directions

  1. Sign into the application using Username and Password.
  2. Click the drop down and select which student you need to access contact information for.
  3. You will then see all the contacts available to you listed on the screen

Login

This web page is the first form the user will see when connected to our Web Service. The user must be logged in in order to be able to make requests.

All fields are required while logging in.

The “UserName” and "Password" text boxes should be populated with only the credentials of the user performing the query.

JSON

JSON or JavaScript Object Notation, is a lightweight text‐based open standard designed for human‐readable data interchange. It is derived from the JavaScript scripting language for representing simple data structures and associative arrays, called objects.

By choosing JSON in the drop down tab under "Output Formats Available" gives the information back in JSON format. When the request is made a structure with an ID, First Name and Last Name of a student will return the details of the contacts linked to that student. The HTTP response will contain a valid status (200 “ok”) in JSON.

Requests

Once the user logs into the service:
*(each user should be given log in credentials from the daycare for employment)*

A form allows the user to access all services from a single GUI.

This service extracts all the students, finds the particular student you entered, grabs each of his/her contacts and outputs their information in a table format to the webpage.

The application will be accessible only to employees through a standard web browser simply by accessing one of the URL's given through employment at Kiddie Kollege Daycare.

In order to use the service in other environments you will require a unique access key that you can get through through the school's IT department. The format for the request must be "http://18.224.74.193:8080/StPaulOnlineEnrollment/services/students/[your-unique-key]/" will return a list of all students. Adding the ID of an individual student after the final slash will return the contact information for the students contacts.
